# 🌍 AI Language Translator

An AI-powered Language Translation Tool developed using **Python**, **Streamlit**, and **Google Translator API** as part of the **CodeAlpha Artificial Intelligence Internship**.

---

## 📌 Project Overview

This application translates text instantly between multiple languages using AI-powered translation services. It provides an intuitive and user-friendly interface with additional features such as text-to-speech, translation history, language swapping, and downloadable translations.

---

## ✨ Features

* 🌐 Translate between multiple languages
* 🔄 Swap source and target languages
* 🔊 Text-to-Speech for translated text
* 📥 Download translated text
* 🕒 Translation history
* 📊 Translation statistics
* 🎨 Modern Glassmorphism UI
* 🖼️ Custom background and logo
* ⚠️ Error handling for invalid inputs

---

## 🛠️ Technologies Used

* Python
* Streamlit
* deep-translator
* Google Translate
* gTTS (Google Text-to-Speech)
